Media Queries

Las media queries se usan para cambiar los estilos de uno o varios elementos dependiendo de el tamaño de pantalla

Este div cambia de color dependiendo del tamaño de la pantalla
  • Se ve de color aqua en un computador
  • Se ve de color amarillo en una tablet
  • Se ve de color violeta en un celular
